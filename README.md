# My Tailor Project

A tailoring management web application built using Flask, SQLite, HTML, CSS, and JavaScript.

## Features

* Customer Registration & Login
* Tailor Dashboard
* Order Management System
* Measurement Tracking
* Admin Dashboard
* Phone Validation
* Responsive Frontend Design

## Technologies Used

* Python Flask
* SQLite
* HTML5
* CSS3
* JavaScript

## Project Structure

```txt
MY_TAILOR/
├── Backend/
├── Database/
├── FrontEnd/
├── Images/
├── README.md
└── requirements.txt
```

## Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/theharshaltech/MY_Tailor-Project
```

### 2. Install Requirements

```bash
pip install -r requirements.txt
```

### 3. Run Flask App

```bash
python app.py
```

### 4. Open Browser

```txt
http://127.0.0.1:5000
```
